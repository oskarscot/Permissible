# Permissible

An example project that can build and run plugins for the game Hytale!

> **⚠️ Warning: Early Access**    
> The game Hytale is in early access, and so is this project! Features may be
> incomplete, unstable, or change frequently. Please be patient and understanding as development
> continues.

## What is permissible?

Default Hytale permission management isn't great, it's messy and not very user friendly. It is however very customizable, `Permissible`
aims to replace the default permission system supported by database persistence and the ECS mechanics.
